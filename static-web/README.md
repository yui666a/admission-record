# 参拝者情報集計
エクセルに記録している教会参拝者の集計を行う
# Usage
1. buildディレクトリのindex.htmlを開く
1. 記録したexcelファイルを読み込ませる

# Features
以下を表示可能
- １日の参拝者数
- 月の参拝者数
  - 男女別
  - グループ別
  - 年代別
- 前年同月との比較
  - 男女別
  - グループ別
  - 年代別

# Note
- ログの量が多くなった場合の速度に関しては計測していません．
